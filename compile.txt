1) replace driver code:

	replace original stmmac driver drirectory drivers/net/ethernet/stmicro/stmmac
	with provided one

2) replace header files:

	replace required headers:

		include/linux/stmmac.h
		include/net/tc_act/tc_gact.h

3) enable the following kernel configs by "bitbake -c menuconfig linux-s32":

	CONFIG_STMMAC_ETH=m
	CONFIG_STMMAC_PLATFORM=m
	CONFIG_DWMAC_S32CC=m

4) modify DTS file arch/arm64/boot/dts/freescale/s32-gen1.dtsi

	as described in backport.txt

5) after kernel boot, load driver manually ("modprobe dwmac-s32cc")


That's it!
